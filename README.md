## Installation (local development)

1. Clone the repo

```bash
git clone https://github.com/mvikashsaini/Vaptro-Ecommercre-Frontend.git
cd <repo-name>
```

2. Install dependencies

```bash

# or
pnpm install

```

3. Start the dev server

```bash

```
